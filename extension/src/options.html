PDF Editor options

Cloudflare URL Scanner (optional)

Links in a PDF are disabled by default. When you enable them in the editor, each URL can be checked with the Cloudflare URL Scanner and rated 🟢/🟡/🔴. Leave these blank to use the offline rating only.

Native host

Document processing runs in a local native host (C#/.NET). Status: checking…

If the host is not connected:

# Linux / macOS
./scripts/install-host.sh <your-extension-id>

# Windows (PowerShell)
.\scripts\install-host.ps1 -ExtensionId <your-extension-id>

Your extension ID is shown at chrome://extensions with Developer mode on. Full instructions are in the repository README.
